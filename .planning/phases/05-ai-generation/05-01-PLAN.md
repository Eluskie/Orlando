<objective>
Build the generation backend: prompt builder that injects brand style into user prompts, generation DB queries, and upgrade the existing /api/generate route to support mock multi-image generation (2-4 variations) with real Imagen 4 integration (behind feature flag). The API also accepts an optional image input for image-to-image/sketch-to-image generation (GEN-02).

Purpose: This is the core generation engine. Without this backend, no UI can trigger image generation. It transforms the existing stub route into a fully functional generation pipeline that retrieves brand style, constructs enhanced prompts, generates multiple variations, stores results to R2, and creates database records. The optional image input enables sketch/reference-based generation.

Output: Working API endpoint that accepts {prompt, brandId, count, aspectRatio, image?} and returns generated asset URLs. Mock mode produces SVG placeholders (ignoring image input); real mode calls Imagen 4 via AI SDK, using image as additional context when provided.
</objective>
